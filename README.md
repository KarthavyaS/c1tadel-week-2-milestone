## Overview

Document Vault is a cross-platform Flutter application designed to securely store, organize, and manage your personal and professional documents. The app offers folder and document management, file protection, and note-taking features with a clean and user-friendly interface.

---

## Features

- User authentication with demo credentials  
- Folder organization with document counts and protection locks  
- Document grid view with descriptive icons  
- Inline note editing and file locking features  
- Responsive design supporting web, desktop, and mobile  
- Single codebase for multiple platforms using Flutter
